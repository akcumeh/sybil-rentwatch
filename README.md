# RentWatch

A **Psycho-Pass / Sibyl System** inspired frontend for the RentWatch hackathon project.

## Installation

Because the scaffolding was done manually into this folder, please run the following to download dependencies and launch the system:

```bash
npm install
npm run dev
```

## Features
- **Strict Dark Mode:** Hardcoded #070B14 colors for the void-like clinical UI.
- **Typography Suite:** 
  - `Zen Antique` for headings
  - `Syne` for body/labels
  - `JetBrains Mono` for live data/scores
- **Count-Up Mechanics:** Integrated Framer Motion count-ups upon load
- **Data UI:** Customized Recharts components featuring neon-blue gradients.
