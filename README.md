# sybil-rentwatch
RentWatch: Enyata x Interswitch Hackathon entry by Team Sibyl

RentWatch is a mutual accountability platform for Lagos real estate. It gives landlords and tenants a shared trust infrastructure that the market has never had: verified identities, escrow-protected payments, and a two-way reputation score called the Hue Score.

## Setup instructions
- Clone this repository
- Navigate to this repo: `cd path/to/sybil-rentwatch`
- navigate to the `data` folder: `cd `
- navigate to `frontend` folder: `cd frontend/`
- copy over environment variables: `cp ../.env.example .env`. make sure to grab and fill out your variables from the various dashboards.
- install relevant packages and run in dev mode: `npm i; npm run dev`
- visit [https://localhost:3000/](https://localhost:3000/) and explore.
